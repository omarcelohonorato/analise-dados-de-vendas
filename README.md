# Análise de Dados em Python – Vendas de Varejo

Este projeto foi desenvolvido como parte da disciplina **Linguagem de Programação** (ADS) e consiste na análise e visualização de dados de vendas de uma empresa de varejo utilizando Python.

## Objetivo
Analisar dados de vendas de 2023 para identificar padrões, calcular métricas por categoria e gerar visualizações que auxiliem na tomada de decisão.

## Tecnologias Utilizadas
- Python
- Pandas
- Matplotlib
- Seaborn
- SQLite

## Funcionalidades
- Conexão com banco de dados SQLite e criação da tabela de vendas.
- Leitura e tratamento de dados com Pandas.
- Cálculo do **total e média de vendas por categoria**.
- Cálculo do **total de vendas por mês**.
- Criação de gráficos: barras, linha e histograma para visualização de resultados.
- Geração de insights sobre categorias mais lucrativas e sazonalidade de vendas.
